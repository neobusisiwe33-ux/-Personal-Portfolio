# -Personal-Portfolio

A fully responsive personal portfolio website built with HTML, CSS and Bootstrap 5.

Full-screen background image with a dark gradient overlay for readable text
Two-column layout — text content on the left, profile photo on the right
Headline: "I am Neo Motsoeneng" with animated role sub-headline
Two CTA buttons: View My Work and Hire Me
Bouncing scroll indicator arrow at the bottom

Sticky Navbar
Fixed to the top — always visible while scrolling
Logo left, navigation links right (Home, About Me, Projects, Contact)
Active underline updates automatically as you scroll between sections
Collapses into a hamburger menu on mobile
About Me
Two-column white card with shadow and rounded corners
Left column: profile photo, name, role, phone, email and skill badges
Skill badges: HTML, CSS, JavaScript, Bootstrap
Projects Grid
3 columns on desktop, 2 on tablet, 1 on mobile
3 real projects: Tesla Landing Page, Netflix Landing Page, YouTube Homepage
Each card has a banner image, tech tag badge, title, description and two buttons — View Website and a code icon button
Cards lift on hover with a zoom effect on the image

Contact Section
Social icons row: Instagram, GitHub, LinkedIn, Email
Clean contact form with fields: Email, Name, Subject, Message and Send button
Form submits directly to neobusisiwe33@gmail.com via mailto
All fields have HTML validation (required)
